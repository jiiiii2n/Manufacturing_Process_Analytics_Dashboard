# Manufacturing Process Analytics Dashboard

## Project Overview

Manufacturing Process Analytics Dashboard는 Python과 Streamlit을 활용하여 제조 공정 데이터를 분석하고 품질을 모니터링하는 웹 기반 대시보드입니다.

공정 데이터를 업로드하면 데이터 전처리, 시각화, SPC 관리도, 공정능력(Cp/Cpk) 분석, 이상 탐지, 수율 분석 등을 수행할 수 있으며, 향후 태양전지 Cell 공정 분석 기능을 추가하여 실제 제조 데이터를 분석하는 플랫폼으로 확장하는 것을 목표로 합니다.
